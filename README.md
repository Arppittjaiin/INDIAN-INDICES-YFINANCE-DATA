# 📈 Nifty & Sensex — Cleaned OHLC Data Dashboard

A streamlined Streamlit application that retrieves, cleans, caches, and visualizes OHLC data for India’s major indices. Designed for reliability, long-term data maintenance, and operational efficiency, it ensures consistent downloads, automated fallback handling, and optimized performance through caching and multithreading.
